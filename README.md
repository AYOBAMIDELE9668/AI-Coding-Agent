# AI-Coding-Agent
A **safe, intelligent, and production-ready AI Coding Assistant** hat can:

- Explain code and programming concepts
- Generate high-quality code in multiple languages
- Improve and refactor existing code
- Debug code using static analysis
- Provide safe recommendations without executing code
- Run as a FastAPI microservice
- Be fully deployable with Docker

---

##  Features

-  Code Explanation
-  Code Generation (Python, JavaScript, Java, Go, SQL, C++, HTML, etc.)
-  Code Improvement / Refactoring
-  Static Debugging (Safe, No Execution)
-  REST API for integration
-  Google Gemini 2.5 Flash LLM
-  LangGraph Workflow
-  Docker Support

---

##  Tech Stack

| Component        | Technology         |
|-----------------|------------------|
| LLM              | Gemini 2.5 Flash  |
| Agent Framework  | LangGraph         |
| Backend API      | FastAPI           |
| Runtime          | Python 3.11       |
| Deployment       | Docker            |


---

##  What This Agent Can Do

- Generate Python, JS, Java, Go, SQL, C++, HTML, and more
- Explain OOP, ML, algorithms, data structures
- Improve existing codebases
- Fix bugs logically
- Provide best-practice suggestions
- Help learn programming in a safe and controlled environment

##  Safety Notes

This Coding Agent is **intentionally built with safety in mind**:

-  Does NOT execute any code
-  Does NOT run shell commands
-  Does NOT modify your system
-  Suitable for minors and educational environments
